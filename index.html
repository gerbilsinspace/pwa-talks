Service Worker

  • A type of 'web worker'
  • Acts as network proxy
  • Event driven (install, fetch etc...)
            
if ('serviceWorker' in navigator) {
  navigator.serviceWorker.register('service-worker.js');
}

Cache API

  • Storage for request / response objects
  • Available directly to the sevice worker
  • Developer responsible for defining and maintaining cache
            
caches.open('showcase-v1').then(cache =>
  cache.addAll([
    'assets/piksel.png',
    'css/main.css',
    'js/main.js'
  ])
);
