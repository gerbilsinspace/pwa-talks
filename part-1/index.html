What are Progressive Web Apps?

  • Just a web 'app'
  • Saved to home screen, no browser chrome
  • Offline caching
  • HTML5 APIs e.g. notifications API

  • Browser decides whether to prompt user to add to home screen

Whats a service worker?

  • A type of 'web worker'
  • Acts as a network proxy
  • Event driven (install, fetch etc...)
            
if ('serviceWorker' in navigator) {
  navigator.serviceWorker.register('service-worker.js');
}

Whats the Cache API?

  • Storage for request / response objects
  • Available directly to the sevice worker
  • Developer responsible for defining and maintaining cache
            
              caches.open('console-v1').then(cache =>
                cache.addAll([
                  'assets/piksel.png',
                  'css/main.css',
                  'js/main.js'
                ])
              );

What should we cache?

  • Application shell (no blank screen :))
  • Dynamic content
